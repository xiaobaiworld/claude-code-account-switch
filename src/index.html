多端共享同步

在 Windows 与 WSL（或两台 LAN 内的机器）各自启动 ccs web，通过 HTTP 互相同步登录凭证。 谁的 OAuth token 更新（expiresAt 更大），就把它推给另一端。
只需一端填对端 URL作为主动方（定时轮询），另一端留空 URL 作为被动方（仅响应请求）。两端密钥必须一致。 凭证走 LAN 明文传输，仅建议同一可信网络使用。
